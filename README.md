# Last Mile Delivery Tracker

## Overview
A MERN Stack web application for managing and tracking last-mile deliveries.

## Features
- JWT Authentication
- Admin Dashboard
- Order Management
- Delivery Agent Management
- Zone Management
- Area Management
- Rate Card Management
- Live Order Tracking
- Analytics Dashboard

## Tech Stack
Frontend
- React.js
- Bootstrap
- Axios

Backend
- Node.js
- Express.js
- JWT
- Nodemailer

Database
- MongoDB Atlas

Deployment
- Vercel
- Render
